Definitions (Group Three)

Authentication is a key part of web security it is used to identify a user connecting to a service to verify the service is not malicious. This can ben done through the use of trusted third party certificates which the service will supply to the user

HTTP is what is used when you try and connect to a web server or local intranet. It establishes the connection to the server initially and once connection is made it then sends back a webpage to the user connecting to it.

Session management is used to establish secure connections between users and the web service they are interacting with. This is done by the user authenticating their id with the web service.

Stateless programming means it does not maintain a state and relies on the input of a value to the function. Meaning the state will change depending on the function input.

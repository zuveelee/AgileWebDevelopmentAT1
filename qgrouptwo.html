Questions (Group Two)

Programming controls are used to create a logical flow of the program based upon certain conditions being met such as the use of an if or else statement to determine the outcome of an action. Programming controls can be designed into four structures sequence, selection, repetition and input-output

Web testing procedures are put in place to help determine if the website is functional and working properly this can be done through a few steps such as functional testing, usability testing, interface testing, compatibility testing, performance testing and security testing. These tests are used to ensure quality work is produced before publishing the website.

To debug a website there are many browser debugging tools that can be used to assist in the process. These tools can be installed in the browser to assist in catching any errors. Tolls such as chrome developer tools or safari developer tools can be used to assist in the debugging process.

Some good techniques or best practices for coding a website are:
Keep it simple – simplify code to make it long and complex.
Meta tags- use descriptive meta tags.
Comments – don’t over comment keep comments simple and precise.
